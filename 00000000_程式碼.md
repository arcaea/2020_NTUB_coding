停一下執行
----
```java
Thread.sleep();
//例如
Thread.sleep(3000); // 暫停3秒
```
說明:Java的Thread.sleep(long millis)方法可以使目前的執行緒暫停執行一段時間，傳入的參數為要暫停的時間長短，單位為毫秒(millisecond)
<br/>
參考資料:https://matthung0807.blogspot.com/2017/09/java-threadsleep.html
