亂數
----
求 0~9 之間的亂數
```java
int r = 0;
r = (int)(Math.random()*10);
```
求 1~100 之間的亂數
```java
int r = 0;
r = (int)(Math.random()*100)+1;
```
停一下執行
----
```java
Thread.sleep();
```
說明:Java的Thread.sleep(long millis)方法可以使目前的執行緒暫停執行一段時間，傳入的參數為要暫停的時間長短，單位為毫秒(millisecond)
Thread.sleep(3000); // 暫停3秒
